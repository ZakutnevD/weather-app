# 🌤️ Погодное приложение

Интерактивное веб-приложение для просмотра текущей погоды и прогноза на 5 дней.  
Использует **OpenWeatherMap API**, **геолокацию**, **localStorage** и динамическую смену фона.



> ⚠️ Замените `YOUR_USERNAME` на имя вашего аккаунта GitHub.

## 📋 Функционал
- 🔍 Поиск погоды по городу
- 📍 Автоопределение местоположения (геолокация)
- 🌡️ Текущая погода: температура, ощущается как, влажность, ветер, давление
- 📅 Прогноз на 5 дней (с иконками)
- 💾 Сохранение последнего запроса в localStorage
- 🎨 Фоновое изображение меняется в зависимости от погоды (солнечно, облачно, дождь, снег и т.д.)
- 📱 Адаптивный дизайн (мобильные устройства)

## 🛠️ Технологии
- **HTML5** – семантическая структура
- **CSS3** – адаптивная верстка, Flexbox, Grid, анимации
- **JavaScript (ES6)** – асинхронные запросы, работа с DOM, Geolocation API, LocalStorage
- **OpenWeatherMap API** – получение данных о погоде
